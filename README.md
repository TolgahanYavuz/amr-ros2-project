# amr-ros2-project
Autonomous mobile robot built with ROS2 &amp; Gazebo — SLAM, Nav2 navigation, and computer vision, developed simulation-first with planned Raspberry Pi deployment.

## Getting Started
### Requirements
- Windows 10/11 with WSL2 support
- VS Code + Remote WSL Extension

### Installation
1. Install WSL2 with Ubuntu 22.04 — [Microsoft WSL Guide](https://learn.microsoft.com/en-us/windows/wsl/install)
2. Install ROS2 Humble Desktop — [Official ROS2 Documentation](https://docs.ros.org/en/humble/Installation.html)
3. Install Git and configure your credentials
4. Clone this repository:
```bash
git clone https://github.com/TolgahanYavuz/amr-ros2-project.git
cd amr-ros2-project
```
